# Graasp App: Text Analysis

Create a `.env.development` file with:

```dotenv
VITE_PORT=3005
VITE_GRAASP_APP_KEY=<app KEY>
VITE_ENABLE_MOCK_API=true
VITE_API_HOST=<request address for the backend>
```
